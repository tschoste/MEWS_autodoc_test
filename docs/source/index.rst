MEWS Documentation
===================

.. figure:: /images/logo.png

Multi-scenario Extreme Weather Simulator (MEWS) is a python-based software developed to predict the increased frequency and severity of various natural phenomena including: heat waves, cold snaps, and eventually hurricanes. This repository is simulating the auto-documentation feature of WNTR.
